Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM. Such as:
1. getElementsByTa Name(): it selects all elements with a specified tag name
2. getElementsByClassName(): it selects all elements with a specified class name
3. getElementById(): it selects a single element by its unique ID
4. querySelectorAll(): it selects all elements that match a specified CSS selector
5. querySelector(): it selects the first element that matches a specified CSS selector

Question-2: What is the difference between innerHTML, innerText, and textContent?

The innerHTML property can read both the HTML markup and the text content of the element. This means when it is use to set the content of elements, it can include HTML tags, and the browser render them correctly.
When innerText is used to read the content of an element, it returns the text as it appears on screen.
The textContent property also ignores all HTML tags and returns only the text. But innerText reads text as it is shown on screen but textContent reads text as it is in the markup which means it returns hidden text also.

Question-3: What is event delegation in the DOM?

Event delegation is a useful technic in JavaScript that takes advantage of event bubbling to handle events better. Instead of adding an event listener to each child element, using delegation in one of the parent element, its children can be managed.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event on an element also triggers on its parent and ancestors, moving up the DOM tree to its root.
